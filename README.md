# LexiLearn 🌟

LexiLearn is a specialized educational web application designed to support children with dyslexia. It provides an engaging, interactive, and modern platform for early screening, skill development, and progress tracking.

## 🚀 Mission

Our mission is to make learning accessible and fun for children with dyslexia by using tailored educational games and a comprehensive screening system, helping parents and educators identify needs early and track progress effectively.

## ✨ Key Features

### 🔍 7-Day Dyslexia Screening Test
A structured, week-long evaluation designed to identify specific learning patterns and areas where a child might need additional support.

### 🎮 Interactive Learning Games
- **Phonics Fun**: Interactive exercises to improve phonological awareness.
- **Vocabulary Builder**: Word-association and spelling challenges.
- **Letter Maze**: Visual-spatial tasks to help with letter recognition.
- **Shadow Matching**: Enhancing visual perception and shape recognition.
- **Story Sequencing**: Developing logical thinking and narrative understanding.
- **Combination Tasks**: Multi-sensory learning activities.

### 📊 Real-time Dashboard
A comprehensive view of the student's progress, showing scores, daily challenge streaks, and overall performance in various categories.

### 👨‍👩‍👧‍👦 Parent's Guide & Analyst View
- **Parents Dashboard**: Guidance and tips for supporting your child's learning journey.
- **Analyst View**: Detailed breakdown of screening results and historical performance data.

### 🏆 Daily Challenges (Tracer)
Engaging daily task like letter tracer to encourage daily practice and maintain learning momentum.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Backend & Database**: Firebase (Real-time Database & Authentication)
- **Design**: Modern, dyslexia-friendly UI/UX with consistent themes.

## ⚙️ Setup Instructions

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Indhu375/LexiLearn.git
   ```

2. **Configure Firebase**:
   - Create a project on [Firebase Console](https://console.firebase.google.com/).
   - Enable Authentication and Real-time Database.
   - Update `firebase-config.js` with your project's credentials.

3. **Run Locally**:
   - Open `index.html` in any modern web browser.
   - For the best experience, use a local server (e.g., Live Server extension in VS Code).
